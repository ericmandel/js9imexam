JS9 Imexam: browser-based imexam using JS9 plugins

You can add browser-based imexam analysis to a JS9 Web page!

In this example, ImExam div elements are defined in the Web-page
  • CAS-A (Chandra)
  • CTB 109 (Einstein)
  • m13 (via SkyView)
  • i1000 (v=x*1000+y)
Load an image, create a region, and move it around to see the imexam windows update. You also can select one or more light imexam windows from the View menu to see other cuts of the data.

In this example, no Plugin div element has been defined on the page. It therefore will be offered in the View menu as a light window.
  • CAS-A (Chandra)
  • CTB 109 (Einstein)
  • m13 (via SkyView)
  • i1000 (v=x*1000+y)
Load an image, create a region, select one or more imexam windows from the View menu. Then move the regions around to see the imexam windows update.
